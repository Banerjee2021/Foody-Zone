
## Deployment

To deploy this project :

1. Change directory to server
```bash
  cd server
```
2. Install all the necessary dependencies
```bash
  npm i
```
3. Start the server
```bash
  npm start server
```
4. Move out of the directory
```bash
  cd..
```
5. Now, change directory to app
```bash
  cd app
```
6. Install all the necessary dependencies for app.
```bash
  npm i
```
7. Run the app
```bash
  npm run dev
```
